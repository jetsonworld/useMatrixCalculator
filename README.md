# 젯슨나노에서 행렬계산기 써보기
***
젯슨 나노 개발보드에서 주피터노트북 기반 행렬(Matrix)계산기가 된다면 얼마나 좋을까요? 행렬은 과학곳곳에서 쓰입니다. 예를 들면, 일차방정식, 미분방정식, 행렬 미적분학, 전기자기학, 양자역학, 경제학, 머신러닝, 딥러닝에 이르기까지 다양한 분야에 응용되어집니다.
